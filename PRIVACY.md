# Privacy Policy

Effective date: April 30, 2026

This Privacy Policy explains what FarDing's Friendship Points collects and how that information is used.

## Information Collected

FarDing's Friendship Points may store the following information when used in a Discord server:

- Discord server IDs
- Discord user IDs
- Channel IDs needed for settings or scoring
- Message IDs when needed for reaction or anti-abuse tracking
- Point totals, weekly totals, profile stats, titles, and administrative adjustments
- Voice participation timing needed to calculate voice points
- Configuration set by server administrators

The bot does not intentionally store full message content. It may inspect message content while running to classify activity, suppress spam, and apply scoring rules, but message content is not saved as a permanent record by the bot.

## How Information Is Used

Collected information is used to:

- Award and track friendship points
- Show leaderboards, profiles, stats, and weekly recaps
- Apply anti-abuse rules such as cooldowns and duplicate suppression
- Support server administrator settings and moderation of bot features
- Maintain, debug, and improve the bot

## Data Sharing

Bot data is not sold. Data is not shared with advertisers. Data may be visible inside the Discord server through bot commands, leaderboards, profiles, stats, recaps, and administrator tools.

Data may be processed by Discord as part of normal Discord platform operation.

## Data Retention

Data is retained while the bot is active in a server or as needed for operation, maintenance, backups, debugging, or abuse prevention. Server data may be deleted if the bot is removed from a server or if a server administrator requests removal.

## Data Removal

Server administrators may request deletion of data associated with their server. Individual users may contact the bot owner or a server administrator to request removal of data associated with their Discord user ID, subject to technical and administrative limitations.

## Security

Reasonable care is used to protect stored bot data. No system can be guaranteed perfectly secure.

## Children's Privacy

The bot is intended for use within Discord servers and is subject to Discord's own age requirements and policies. The bot is not intended to knowingly collect information from children outside of normal Discord platform use.

## Changes

This Privacy Policy may be updated from time to time. Continued use of the bot after changes are posted means you accept the updated policy.

## Contact

For privacy questions or deletion requests, contact the bot owner or the administrator of the Discord server where the bot is installed.
